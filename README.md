# 📊 Proyecto ENA – Análisis de Unidades Agropecuarias del Perú

El análisis se basa en la información oficial de la Encuesta Nacional Agropecuaria (ENA) del INEI, con el objetivo de evaluar la distribución, características productivas y actividades económicas de las Unidades Agropecuarias en el Perú.

Se realizó la limpieza, depuración y transformación de datos en R para garantizar consistencia y calidad en los indicadores. Posteriormente, se desarrolló un modelo de datos y un dashboard interactivo en Power BI que permite analizar KPIs, comparar actividades agrícolas y pecuarias y visualizar la información por departamento mediante mapas geográficos y segmentaciones dinámicas.

🔧 Herramientas utilizadas: R, Excel, Power BI, Power Query y DAX para limpieza, modelado de datos, visualización y creación de métricas personalizadas.
