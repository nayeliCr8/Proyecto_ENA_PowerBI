# Proyecto_ENA_PowerBI
Dashboard interactivo en Power BI para analizar la distribución, actividades y productividad de las Unidades Agropecuarias del Perú con datos del ENA (INEI). Incluye limpieza y tratamiento de datos, diseño de KPIs, mapa geográfico y comparación de actividades agrícolas y pecuarias para apoyar la toma de decisiones.
