# 📊 Proyecto ENA – Análisis de Unidades Agropecuarias del Perú

El análisis se basa en la información oficial de la Encuesta Nacional Agropecuaria (ENA) del INEI, con el objetivo de evaluar la distribución, características productivas y actividades económicas de las Unidades Agropecuarias en el Perú.

Se desarrolló un dashboard interactivo que permite visualizar indicadores clave, comparar actividades agrícolas y pecuarias, y analizar la información por departamento para apoyar la toma de decisiones basadas en datos.

🔧 Herramientas utilizadas: Power BI, Power Query y DAX para limpieza, modelado de datos, visualización y creación de métricas personalizadas.
